# Nx


## Setup

```bash
npx nx@latest init
```


## Usage

```bash
./nx build my-project
./nx generate application
./nx graph
```

### Custom commands

```bash
./nx run service-a:install

./nx install service-a

./nx test service-a

./nx build service-a
```
